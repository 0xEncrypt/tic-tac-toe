# MERN Tic-Tac-Toe Multiplayer 🎮

A full-stack, real-time multiplayer Tic-Tac-Toe game built with the MERN stack (MongoDB, Express, React, Node.js) and Socket.io. 

This project features a responsive dark-mode UI, an unbeatable AI algorithm, room-based online matchmaking, and a persistent match history database.

## ✨ Features

* **Single Player (AI):** Play against an unbeatable computer opponent powered by the Minimax algorithm.
* **Local Multiplayer:** Play with a friend on the same device.
* **Online Multiplayer:** Create or join a room using a custom code to play in real-time across different devices using Socket.io.
* **Match History:** All completed games are securely saved and displayed on a global leaderboard using MongoDB.

## 🛠️ Tech Stack

* **Frontend:** React (Vite), CSS3 (Animations, Media Queries, CSS Variables), Axios, Socket.io-client
* **Backend:** Node.js, Express.js, Socket.io
* **Database:** MongoDB, Mongoose

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed and a [MongoDB URI](https://www.mongodb.com/) (either local or MongoDB Atlas).
